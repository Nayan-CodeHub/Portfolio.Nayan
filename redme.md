# Portfolio.Nayan

A personal portfolio website for **Nayan Gharat**, showcasing skills, projects, education, and contact details.

## Description

This is a static HTML/CSS/JavaScript portfolio website built with:

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- Google Fonts
- Particles.js background
- AOS scroll animations

## Pages / Sections

- Home
- About
- Skills & Technologies
- Education
- Projects
- Contact

## Usage

Open `index.html` in your browser or serve the folder from a local web server.
