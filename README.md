# Portfolio.Nayan

A modern personal portfolio website for **Nayan Gharat**, featuring an elegant UI, animated sections, project cards, a theme toggle, a contact form, and an admin panel for managing messages.

## Features

- Responsive landing page
- About, skills, education, and project sections
- Animated project cards and hero text
- Light/dark theme toggle
- Contact form with PHP backend
- Admin page to view and delete saved messages
- Resume page

## Tech Stack

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Font Awesome
- Google Fonts
- Particles.js
- AOS

## Run Locally

### Static preview
Open the project folder in a browser, or run:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000/
```

### PHP + MySQL setup
For the contact form and admin features:

1. Start Apache and MySQL in XAMPP/WAMP.
2. Import the SQL from [setup_db.sql](setup_db.sql).
3. Open the project from your local web server.
