# Deployment Notes

## Quick local preview
Run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```

## PHP + MySQL setup
For the contact form and admin page, use XAMPP or WAMP and place the project in the web root folder.

1. Start Apache and MySQL.
2. Import the SQL from setup_db.sql.
3. Open the project in your browser.
